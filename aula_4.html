Aula 4 – Funcionalidades do Sistema

Gerenciamento de Arquivos

▶ Excluir arquivos - Selecionar item e pressionar Delete
▶ Restaurar arquivos - Acessar Lixeira e usar "Restaurar"
▶ Recuperar permanente - Usar software especializado

Lixeira / Exclusão permanente de arquivos

  1. Clique direito na Lixeira
  2. Selecione "Esvaziar Lixeira"
  3. Confirme a ação

Atenção: Esta ação é irreversível!

Desinstalar Programas

▶ Acessar Painel de Controle
▶ Selecionar "Programas e Recursos"
▶ Localizar o programa desejado
▶ Clicar em "Desinstalar"

Painel de Controle

Configurações Úteis:

  • Aparência do Sistema
  • Opções de Teclado
  • Segurança Básica

Acesso Rápido:

  • Tecla Windows + X
  • Menu Iniciar → Configurações
  • Barra de pesquisa: "Painel"

Dica Importante

Sempre verifique o que está sendo excluído
Mantenha backups importantes
